# 项目的构建
1. 项目构建的方法是：asembly/bin/supersonic-build.sh 脚本可以构建 java 和 webapp 端。构建时间很长，你需要耐心等待一下。

# 网络结构
2. 你所在的服务器的IP地址是 10.0.12.244，通过vpn连接的。

# 数据库信息
3. supersonic 的数据库信息你可以在 .env 文件中读到，业务数据库信息存在 supersonic 的数据库中，你自己去读。

# 所关联的项目的源码
4. supersonic的官方源码位置在 /raid/ai/metric/supersonic 中，你可以用来对比参考。
5. superset的版本是6.0.0， 它的调用入口在 .env 中也有， 源码在/raid/ai/metric/superset-6.0.0，使用的是docker启动方式。
6. /raid/ai/metric/coding 目录中有 本项目 supersonic、dbt-mcp、mf2s2-mcp、metricsys_ai 4个关联项目。
